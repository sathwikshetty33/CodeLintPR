üìå How It Works

This tool allows you to analyze a GitHub pull request by providing the following:

  • Repository URL (e.g., https://github.com/user/repo)
  • GitHub Token with repo scope
  • Pull Request Number
üîê Using the Auth Token

The auth token is stored securely in your browser‚Äôs sessionStorage. This token is required in the header of every request to access protected endpoints like:

POST /start-task/
GET /task-status/<task_id>/

Include it like this:

{
  "Authorization": "token <your_token>",
  "Content-Type": "application/json"
}
